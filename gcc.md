# Dicas de gcc para Linux


 

## compilar programa em C

$  gcc    `nome_do_fonte.c`  -o  `nome_do_execeutável`   




## compilar programa em C com pthread
 

$ gcc  `nome_do_fonte.c`  -o  `nome_do_execeutável`    -lpthread




## executar
 
$  ./p
